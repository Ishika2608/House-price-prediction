# 🏠 House Price Prediction

A beginner-friendly end-to-end machine learning project that predicts California house prices using **Linear Regression** and **Random Forest**.

---

## 📌 Project Overview

| Detail | Info |
|--------|------|
| **Dataset** | California Housing (built into scikit-learn) |
| **Problem type** | Regression (supervised learning) |
| **Best model** | Random Forest Regressor |
| **R² Score** | ~0.82 |
| **MAE** | ~$22,000 |
| **Tech stack** | Python, Pandas, Scikit-learn, Streamlit |

---

## 📁 Files

| File | Description |
|------|-------------|
| `house_price_prediction.py` | Main ML script — EDA, training, evaluation |
| `app.py` | Streamlit web app for live predictions |
| `requirements.txt` | All dependencies |

---

## 🚀 How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Train the model and generate plots
python house_price_prediction.py

# Launch the web app
streamlit run app.py
```
